=============
hostedpi list
=============

List all Pis in the account

Synopsis
========

.. code-block:: text

    hostedpi list [-h]

Description
===========

.. program:: hostedpi-list

.. option:: -h, --help

    Show this help message and exit

Usage
=====

List all Pis in the account:

.. code-block:: console

    $ hostedpi list
    mypi
    mypi2
    mypi3
    mypi4
