=============
hostedpi list
=============

.. program:: hostedpi-list

List all Pis in the account

.. code-block:: text

    optional arguments:
      -h, --help  show this help message and exit

Usage
=====

List all Pis in the account:

.. code-block:: console

    $ hostedpi list
    mypi
    mypi2
    mypi3
    mypi4
